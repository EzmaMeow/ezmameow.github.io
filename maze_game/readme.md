# Maze Game Example

This is a test web project that create a 3d maze from a image with minimal dependencies.
It is subjected to change as longs as it is refer to as a test, experiment or prototype.


# Dependencies:

-Three.js

Currently remapping three.js so it dose not need to be included, but require internet acess.


# Modules

-game_ultility:

Three.js dependent function library dealing with math and vector logic.

-game_core:

Three.js dependent base classes for the game.

-maze:

The main file for the maze game. It implements all the game logic and content.
